# Leap Scholar IELTS Preparation Prototype

Interactive product prototype built as part of the Product Intern assignment.

## Features

- Adaptive daily study plans
- IELTS score forecasting
- Weekly progress visualization
- Recovery mode after inactivity
- Skill-wise personalization
- Interactive demo lessons
- Live deployment via Vercel

## Tech Stack

- React + Vite
- Tailwind CSS
- LocalStorage demo backend
- SVG graphs for analytics

## How to Run Locally

```bash
npm install
npm run dev
